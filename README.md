## About

- Slutter p√• toppen is a comprehensive tool for optimizing fantasy football teams using statistical analysis and machine learning. The project aims to help users make data-driven decisions in selecting the best players for their fantasy football teams, considering various factors such as player performance, match difficulty, and budget constraints.

- Python 3.8 or higher
- Pandas
